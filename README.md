## What's in here

One self-contained `index.html`. No build step, no dependencies to install.

- [Three.js](https://threejs.org/) — the 3D snoofi pup (built from primitives, no model files)
- [GSAP ScrollTrigger](https://gsap.com/) — scroll-driven choreography
- [Lenis](https://lenis.darkroom.engineering/) — smooth scrolling
- All loaded from CDN

Scroll the page and the pup drifts, spins, and comes in for a close-up between sections. Click him — he likes it.
